# Phlow VS Code extension (minimal)

Esta extensão mostra uma mensagem "Olá — Phlow: hello world!" toda vez que um arquivo `.phlow` for aberto no editor.

Como testar:

1. Abra este repositório no VS Code.
2. Vá para a aba Run and Debug.
3. Escolha a configuração "Run Phlow Extension (examples)" e inicie.
4. A nova Janela do Extension Development Host abrirá com a pasta `examples` carregada — abra qualquer arquivo `.phlow` e verá a mensagem.

Observação: este é um scaffold mínimo; adicione comandos, contribuições e linguagem conforme necessário.
